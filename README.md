# EON Theme

A system theme based on [this theme](https://github.com/diinki/diinki-retrofuture) with a modified color pallete.

The color pallete is derived from the following wallpaper of the Eon Duo.

![alt text](./wallhaven-9dm39x.jpg)

Arch: system themes should be placed under `~/.local/share/themes/`
